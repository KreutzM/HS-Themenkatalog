# Themenpflege im Alltag

## Empfohlene redaktionelle Regeln

### Titel
- möglichst präzise,
- nicht zu breit,
- möglichst mit technischem Fokus.

### Metadaten
Jede Themenkarte sollte mindestens sinnvoll ausfüllen:
- `difficulty`
- `type`
- `prerequisites`
- `practice`
- `owner`
- `updated`
- `tags`

### Leitfragen
Leitfragen sollen keine bloßen Stichworte sein, sondern echte Bearbeitungsfragen, die Präsentation und Ausarbeitung strukturieren helfen.

### Abgrenzung
Gerade im 4. Semester ist eine klare thematische Abgrenzung wichtig. Themen sollten so beschrieben sein, dass die Studierenden erkennen, was **nicht** Teil der Bearbeitung ist.

## Qualitätscheck vor Freigabe

Vor dem Setzen auf `active` sollte geprüft werden:

- Ist der Fokus realistisch für 30 Minuten Vortrag + Ausarbeitung?
- Sind genug belastbare Quellen vorhanden?
- Ist das Thema für Studierende verständlich formuliert?
- Ist der Praxisanteil nur ergänzend und nicht zwingend vorausgesetzt?
- Ist das Thema klar von ähnlichen Karten abgegrenzt?

## Empfehlung für semesterweise Pflege

- **nach Vergabe:** vergebene Themen auf `used`
- **bei größeren Überarbeitungen:** vorübergehend `hidden`
- **bei neuen Ideen:** zunächst `draft`
- **für den veröffentlichten Katalog:** nur freigegebene Themen auf `active`
